<h1>Formula 1 tracking app</h1>
<p>It shows closest track and it's map also 2 Ferrari racers: Leclerc and Hammilton. Last block shows static info about Scuderia Ferrari team</p>
<p>This project built with OOP principles and built for showing it in university. No more</p>

<h2>To build app:</h2>
<p>Insert this commands in IDE terminal</p>
<code>yarn i</code><br />
<code>yarn run dev</code>
<p>it's done!</p>
